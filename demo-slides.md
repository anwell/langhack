# LangHack

Voice-first language practice for real conversations.

Pick a scenario. Talk. Get feedback.

---

## The Problem

You studied for years but freeze when someone actually talks to you.

Apps teach words — not conversations.

---

## How It Works

1. Choose a scenario (☕ Order at a café)
2. See your goal ("Order a drink and pay")
3. Have a live voice conversation
4. Achieve the goal → auto-pass
5. Get scored feedback + corrections

## Under the Hood

**3 AI Agents:**
- 🗣️ **Conversation Agent** — Nova Sonic real-time voice roleplay
- 🎓 **Teacher Agent** — Nova Lite post-session scoring + feedback
- ✈️ **Travel Agent** — Apify web scraping → Nova Lite scenario generation

**Integrations:** AWS Translate · Box cloud backup · Kiro IDE
**Stack:** Strands BidiAgent · FastAPI · React Native / Expo

---

## Key Features

🎯 Goal-based scenarios with auto-pass
💬 Reply suggestions when you're stuck
🌐 Live English translations (optional)
📊 Score, corrections, vocabulary, lesson plan
☁️ Markdown session reports → Box
